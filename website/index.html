Getting Started

1

Install

# Clone and build git clone https://github.com/your-repo/glados cd glados make
2

Write Code

# hello.gla def greet(name): print("Hello,", name) greet("World")
3

Run

# Execute your program ./glados hello.gla # Output: Hello, World
